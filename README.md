#  Todo App (Flask + SQLite)

A simple and clean Task Manager web application built using Flask and SQLite.
Users can add, update, delete, and mark tasks as completed.


##  Features

* ➕ Add new tasks with description
* 📋 View all tasks
* ✏️ Update existing tasks
* ❌ Delete tasks
* ✅ Mark tasks as completed (checkbox)


##  Tech Stack

* Python (Flask)
* SQLite
* HTML, CSS


##  Project Structure

todo-app/
│── app.py
│── templates/
│── static/




Open in browser:
http://127.0.0.1:5000/
